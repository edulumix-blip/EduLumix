## 🎨 **Rich Snippets & Featured Snippets**

### **Targeted Rich Snippets:**

1. **Job Listings:**
```json
{
  "@type": "JobPosting",
  "title": "Software Developer",
  "salary": "₹3-6 LPA",
  "location": "Bangalore"
}
```
**Result:** Appears in Google Jobs with salary, company logo

2. **Course Listings:**
```json
{
  "@type": "Course",
  "name": "Full Stack Development",
  "price": "₹999",
  "rating": "4.8"
}
```
**Result:** Shows price, rating in search results
